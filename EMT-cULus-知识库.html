§5.1 管材本体构造要求

规定 EMT 钢管本体的几何与表面完整性，属机械合格性的第一道关。

条款要求项具体规定
5.1.1基本形态必须为钢制、直形、圆形截面，所有焊缝必须完全焊透（fully welded through）。
5.1.2焊缝突起焊缝处不得有金属毛刺、锐边或锐利突起；内壁焊缝允许轻微焊瘤，但高度上限：规格 16–53（1/2–2 in）≤0.38 mm；规格 63–103（2-1/2–4 in）≤0.51 mm。
5.1.3涂覆前表面涂覆保护层之前，内外表面须无氧化皮（scale）、飞边及其他突出物。
5.1.4管端管端必须切割方正（cut square），无毛刺和锐边。
5.1.5螺纹EMT 不得带螺纹，除非整体接头（integral coupling）为钢制且螺纹为非标准管螺纹规格。
速记：焊缝完全焊透 + 内壁焊瘤限高 + 表面无氧化皮 + 端口方正 + 不带标准管螺纹。这五条是焊管制造端最直接的质量控制点。

§5.2 材料 / 化学成分要求

🧱 基材定义

UL 797 §5.2 定义基材为 “碳素结构钢、电阻焊（ERW）”，即低碳软钢。化学成分（熔炼分析 heat analysis）限值在 §5.2 / Table 2 规定，但标准正文付费，未公开具体元素限值。

🔬 等效牌号（实务）

EMT 用钢实务上相当于：
SPCC (JIS) Q195 (GB) ASTM A1011 SS Gr.33
碳含量很低（C 约 0.05–0.15%，上限 ≤0.25%），低锰、无 Cr/Mo 等合金元素。

元素典型范围上限（参考）说明
碳 C0.05–0.15%≤0.25%低碳 → 易冷弯、焊缝韧性好
锰 Mn0.25–0.60%≤0.90%低锰，保证软态成型

关键提示（极重要）：UL 797 对基材【没有】规定明确的最低抗拉/屈服强度数值，也【没有】抗压（crush）、抗冲击（impact）、跌落等机械破坏性试验——这与刚性金属导管（UL 6 RMC）等明显不同。EMT 的机械合格性主要由 弯曲试验 + 尺寸公差 + 焊缝完整性 三者保证。

§5.3 保护涂层要求

EMT 防腐核心，分主涂层、替代涂层、内层有机涂层、附加涂层与薄表面处理五类。

涂层类型位置主要要求
锌涂层 Zinc 外表面（主涂层）/ 内表面均可 完全覆盖、牢固附着、平滑无气泡，与钢材金属接触；须通过硫酸铜试验（§6.2.2）。
替代耐腐蚀涂层 Alternate 外表面 完全覆盖、牢固附着、平滑；须通过 §6.2.4 系列腐蚀试验；非金属涂层默认温度范围 0–90°C，超出需额外标注。
有机涂层 Organic 内表面 均匀覆盖、质量一致、外观平整光滑；须通过 §6.2.3 暖湿空气试验和弯曲试验。
附加涂层 Supplementary 外表面（主涂层之上） 无需满足主涂层腐蚀要求，但须评估阻燃性、对主涂层影响、配件配合性及电气连续性；未评估腐蚀性能的需特别标注（见 §7.7）。
表面处理 Surface Treatment 外表面 厚度 ≤0.0038 mm（0.00015 in）的顶涂或转化涂层（如钝化/铬化膜），无需满足替代涂层或有机涂层要求。

§5.4 弯头（Elbow）要求

项目要求
材料与直管同等级钢材，同等处理和涂层。
角度不得超过 90°，公差 ±2°。
最小尺寸（部分规格）详见 Table 5.2：规格 16 弯曲半径 R=102 mm、直段 Ls=38 mm；规格 27 R=146 mm、Ls=48 mm；规格 103 R=406 mm、Ls=86 mm。
整体接头如带整体接头，须符合相关配件标准（fittings standard）。

§6.1 管材弯曲试验（机械性能核心）

试验条件要求
常温弯曲 6.1.1 最小规格样品可弯成 1/4 圆弧（90°），不得开裂、焊缝不得开裂、截面不得明显变形；常温用标准弯管器，UL 797 未规定芯轴（mandrel）（第9版已删除 mandrel 术语）。
低温弯曲 6.1.2 在 0°C（32°F）下预处理 60 min 后可弯成 1/4 圆弧，管不得开裂、焊缝不得开裂；须在冷室内或取出后 15 s 内完成；测最小三个规格 1/2″、3/4″、1″，绕 Table 5.2 规定芯轴半径弯曲。
适用规格 规格 16/21/27 用 图2（Figure 2）弯管器；规格 27 以上用任意合适弯管设备。
判据本质：弯曲后“管壁不裂 + 焊缝不裂 + 截面不显著扁塌”即合格——这是 EMT 韧性/成型性的唯一硬性力学验证，没有定量强度数值。

§6.2 涂层试验（化学 / 防腐性能核心）

§6.2.2 硫酸铜（Preece）试验 — 镀锌层最关键判据

参数规定
溶液标准按 ASTM A239，比重 SG 1.186 @18°C（64°F），pH 3.0–4.0 @25°C，全程维持 18°C。
外层锌浸泡连续 4 次 × 60 s 浸泡。
内层锌浸泡1 次 × 60 s 浸泡。
合格判据最后一次浸泡后无“光亮、牢固附着的金属铜沉积”。
最易失败项 — 必读：这是 EMT 认证中最容易失败的测试，焊缝区最危险（焊接热影响区镀锌往往偏薄）。判读关键：深色 / 黑色铜斑不算失败，只有光亮、牢固附着的金属铜才算失败——意味着该处锌层已被消耗殆尽、露出钢基。
镀锌重量：按 ASTM A90（剥离称重法）或磁性测厚法测定；UL 797 §6.2.2.5 规定最低值（具体数值在付费正文，未公开）。合格性主要靠硫酸铜试验体现，而非单纯称重——称重达标但焊缝处局部薄仍可能 Preece 失败。

机械 vs 化学要求一览（汇总卡）

🔧 机械性能要求
  • 焊缝完整性（§5.1）— 完全焊透、内壁焊瘤限高、端口方正

🧪 化学 / 防腐性能要求
  • 基材低碳碳素钢成分（§5.2）— 低 C、低 Mn、无合金
  • 锌层硫酸铜 Preece（§6.2.2）— 外 4×60 s / 内 1×60 s

⚠ ZAM 镀层在标准中的定位

当前状态：截至 2026 年，无已知 ZAM 涂层 EMT 的 UL 797 认证案例。

ZAM（Zn + ~6% Al + ~3% Mg）属“锌基 / 替代涂层”，技术上可走 §6.2.4 替代涂层路径（盐雾 / CO₂-SO₂ / UV），但需向认证机构提前工程预评估（engineering pre-assessment）。

针对“要看 CSA 22.2 或 UL 有没有化学成分要求”的疑问——结论：标准对替代涂层是按性能（腐蚀试验）而非按成分判定，因此 ZAM 的合金配比本身不构成成分门槛障碍；但仍需机构确认该涂层体系能否通过 §6.2.4 全套腐蚀试验，并接受工程评估后纳入。

实务建议：承诺 ZAM 方案前，先联系 UL Solutions / Intertek 工程师做预评估（pre-eval），确认 §6.2.4 测试路径与样品要求，避免正式测试阶段被否决，造成模具与排产损失。

① EMT 基材到底要什么钢 UL 797 §5.2 + 实务

EMT（Electrical Metallic Tubing，薄壁电工金属管）是薄壁、可现场手弯、靠镀层防腐的排线导管。UL 797 对管体材质的核心约束很明确，决定了基材必须是"软、纯、易成型、易镀锌"的低碳钢：

🔩 EMT 合格基材的五条硬指标
  • 碳素结构钢（low-carbon steel）：UL 797 明确要求碳素钢，不接受合金钢牌号。
  • 电阻焊（ERW）成型：带钢高速辊压 → 高频电阻焊缝 → 定径，是 EMT 的标准产线工艺，无预热工序。
  • 低碳：C 约 0.05–0.15%，规范上限 ≤0.25%；低锰、无 Cr/Mo/V 等合金元素。
  • 可现场手弯 90°：电工用 Greenlee / Ridgid 手弯器，无回火即冷弯不开裂、不起皱。
  • 镀锌后过硫酸铜试验（Preece test）：连续浸渍考核锌层连续性与厚度，是 UL 797 镀层验收的关键关。

壁厚控制：优质 ERW 焊管厂实测可达 ±0.1mm（已确认无锡某焊管厂能做到这一精度）。商品化价格段，供应链成熟。

方向核对：项目中提出的"材料跟内枪管一样用 SPCC"正是上面这一类低碳软钢——方向正确。SPCC（日标冷轧低碳钢板）、中国 Q195、美标 A1011 SS Gr.33 属同一族，都满足 UL 797 的碳素钢定义与镀锌相容性要求。

② ASTM A519 Grade 4130 是什么 Cr-Mo 合金钢

🛠️
ASTM A519 Grade 4130（AISI 4130 / 30CrMo）
Chromium-Molybdenum 合金结构钢 · 无缝机械管材标准

4130 是经典的 Cr-Mo 合金结构钢，以 0.30% 碳 + 约 1% 铬 + 0.2% 钼为核心成分，可热处理强化，典型用途是高压管路、机械结构件、航空构件、赛车防滚架。

  • ASTM A519 是无缝（seamless）机械管标准——它本身不覆盖 ERW 焊管。
  • 4130 的焊管/冷拔版走 ASTM A513 Type 5（DOM，Drawn Over Mandrel），是先 ERW 焊后冷拔的精密管，价格更高。
  • "合金钢"这一身份本身，就已经踩中 UL 797"必须为碳素钢"的红线。

④ 关键障碍排序 按严重度

1
🔴 热镀锌失效（最致命）
0.80–1.10% Cr 生成致密 Cr₂O₃ 钝化膜，阻碍熔融锌润湿铁基，AGA（美国镀锌协会）实证：含铬合金钢镀锌逐批波动、呈类不锈钢行为，涂层连续性无法稳定保证。EMT 靠镀层吃饭，这一条直接判死刑。
2
🔴 UL 797 钢材定义不符
4130 是合金钢（Cr-Mo），非碳素结构钢。UL 797 对管体材质有明确的碳素钢要求，合金钢牌号在认证层面即不被接受，与镀锌结果无关。
3
🟠 ERW 产线需预热
CE ≈ 0.61 > 0.45 阈值，焊接必须 150–205°C 预热以防焊缝/热影响区淬硬开裂。EMT 高速 ERW 连续产线没有也容不下预热工序，强行上会牺牲速度并引入焊缝缺陷。
4
🟠 手弯阻力过剩
屈服强度过剩约 65%，弯管力同步上升约 65%，标准电工手弯器吃力，违背 EMT"现场易弯"的产品定位。
5
🟡 成本溢价 30–60% 无收益
合金钢 + DOM 工艺使材料成本升至 1.5–1.8×，而 EMT 既不需要高强度也不需要高压性能，溢价完全沉没。

⑤ 碳当量小科普 为什么 4130 要预热

📐 展开：碳当量 CE 怎么算，4130 为何卡在 0.61

国际焊接学会（IIW）碳当量公式：

CE = C + Mn/6 + (Cr+Mo+V)/5 + (Cu+Ni)/15

代入 4130 典型成分（C 0.30 / Mn 0.50 / Cr 0.95 / Mo 0.20）：

CE ≈ 0.30 + 0.50/6 + (0.95+0.20)/5
   ≈ 0.30 + 0.083 + 0.23 ≈ 0.61

  • CE < 0.45：可焊性好，一般无需预热 —— EMT 低碳钢 CE ≈ 0.13–0.18，落在这一区，高速 ERW 直接焊。
  • CE > 0.45：淬硬倾向高，焊接需预热（4130 推荐 150–205°C），否则焊缝/HAZ 易冷裂。

这就是为什么"把 4130 拿来跑 EMT 的 ERW 产线"在工艺上根本对不上——产线没有预热环节，而 4130 又离不开它。

⑥ 给 PrecisePipe 的选材建议 结论

推荐基材：低碳软钢 SPCC / Q195 / Q195L。工艺路线：ERW 焊管 → 去除内外焊缝毛刺 → 250 g/m² 吹镀（thermal diffusion / 内外热镀）。该路线满足 UL 797 碳素钢定义、镀层连续性与硫酸铜试验要求，且现场手弯、成本均处于商品化最优区间。
务必分清两个独立问题：
  • 基材选型（本页）：管体用什么钢 → 低碳钢 ✓ / 4130 ✗。4130 的问题出在基材本身。
  • 镀层选型：表面防腐方案 → 预镀锌 / 250g 吹镀 / ZAM（Zn-Al-Mg）。ZAM 是镀层方案，与基材是不是 4130 无关。
别把这两条轴混为一谈：换 ZAM 镀层救不了 4130 的基材问题，4130 即便镀 ZAM 也仍然过不了 UL 797 的碳素钢定义与 ERW 预热门槛。

两条独立坐标轴示意

两条轴正交：基材决定能否过 UL 797 的"碳素钢 + 可焊 + 可弯"门槛；镀层决定防腐等级与硫酸铜/盐雾表现。选材时分别决策，不要互相牵连。

📐 为什么吹镀过硫酸铜、预镀锌过不了？(技术注解)
硫酸铜浸渍(CuSO₄)试验通过锌层被溶穿后铜析出判定锌层连续性与厚度。预镀锌带钢锌层仅 5–10μm(约 35–70 g/m² 单面)，在切口、焊缝处更薄，硫酸铜数次浸渍即露铁判废。自有吹镀做到 250 g/m²(约 35μm/单面)有充足余量。ZAM(锌铝镁)虽锌当量低，但表面致密氧化膜延缓溶解，实测可过——这是 ZAM 方案的主要卖点，但它属替代涂层，须机构按性能路径单独认可。 详见 🧪 测试要求 与 📋 适用标准。

⚠ 最大风险 — 镀锌路线摇摆

✓ 推荐主线 — 250g 吹镀 · 单厂 · SPCC 基材

💡 五条落地建议
  1. 基材锁定 SPCC：低碳、无 Cr/Mo、镀锌相容好、延性高(易手弯)，与 EMT 基材标准高度吻合。唯一注意：SPCC 属商业级、强度偏软(典型抗拉 270–340 MPa)，略低于 A1011 SS33 的 360 MPa 目标——但 UL 797 本就不设强度门槛(由弯曲+尺寸把关)，必要时改半硬态 / 略高碳热批次即可补强。详见 🔩 钢材选型。
  2. 预咨询前置：投钱打样前先做一次机构免费 / 低成本预咨询，一次问清 ZAM 替代涂层路径、化学成分要求、cULus vs cETLus。
  3. 自测前置：把硫酸铜内部自测放到送实验室之前，避免外部测试返工(返工 = 钱 + 周期双输)。
  4. 单厂优先：尽量找焊管 + 吹镀一体厂，规避双厂多一次初检 + 多一次年度 FUS。
  5. 主线锁定：对外报价与认证主线统一锁 250g 吹镀，ZAM 仅作平行验证，避免供应侧单点推动改变既定路线。

🏭 D. 工厂检查 速查

一句话澄清：UL 体系里没有独立的"厂家认证"——认证结构是 E 号 = 特定产品 + 特定标准 + 特定工厂 三方绑定。所谓"成品认证"就是产品测试通过拿到 E 号；工厂地址被写进该 E 号文件、通过初检并持续接受 FUS，才有权生产带 cULus 标志的产品。换厂 / 新增厂 → 新工厂须重做初检并更新文件。

🔍 工厂初检要点

认证机构派工程师现场审查，4 小时 ~ 1 天（UL Solutions 广深沪有本地工程师）。三类结果：✓ 通过加入文件 / ⚠ 有条件整改复查 / ✗ 不通过整改重检（+费用）。

核查三大块：

  • 设备工艺：ERW 成管线、焊缝处理、镀锌线参数、焊缝区镀锌均匀性
  • QC 设备：外径/壁厚量具校准、铜硫酸盐 Preece 测试设备、锌层重量检测
  • 记录体系：来料检验、在线抽检、不合格品隔离、打标系统、批次可追溯
📅 季度 FUS 跟踪要点
  • 频率：每年 4 次（约每 90 天一次）
  • 方式：不预告 Unannounced
  • 时长：2–4 小时/次

现场从成品库抽样测 OD/壁厚、查 UL 标志印制、审 Preece 在线记录、确认工艺无未申报变更、查 QC 设备校准。
工艺重大变更未申报 → 立即暂停，可能吊销认证。

成品认证 vs 厂家认证：UL 体系的真实结构

📄 UL 认证的实际结构

在 UL 体系中，没有独立的"工厂认证"。cULus 认证结构是：

  • E 号 / 文件号 = 特定产品 + 特定标准 + 特定工厂的三方绑定认证
  • 产品必须通过测试；工厂必须通过初检和持续 FUS
  • 换工厂生产 → 新工厂须重新做工厂初检，加入文件

所谓"成品认证"，就是指产品测试通过并拿到 E 号。UL 认证 = 产品测试 + 工厂绑定，两者不可分离。

🏭 cULus 认证的"厂家"含义

UL 体系中没有独立的"厂家认证"。"认证了工厂"的真实含义是：

  • 该工厂地址被写入 UL 认证文件（E 号），并通过了 UL 初检
  • 该工厂每季度接受 UL Follow-Up Services 跟踪审核
  • 工厂有权生产带 cULus 标志的该 E 号产品

更换工厂地址或新增工厂，须重新做初检并更新文件，才能继续生产认证产品。

确定标记方案——连续印字系统（喷码/滚印），确认标记内容和间距符合 UL 797 §9

  1. CA 团队联系 UL Solutions 广州（或 Intertek 广州 info.guangzhou.gs@intertek.com）做预评估咨询

  4. 向无锡焊管厂询价时同时确认对方是否有配套镀锌能力
  5. 锁定方案后提交正式申请，进入测试流程
